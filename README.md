# sv-lms-kit

**sv-lms-kit** adalah proyek pribadi Learning Management System (LMS) yang dibangun menggunakan **Svelte** dan didukung oleh [`sv`](https://github.com/sveltejs/cli).

---

## Informasi Proyek

| Properti           | Keterangan                                                   |
|--------------------|--------------------------------------------------------------|
| Nama Proyek        | sv-lms-kit                                                   |
| Jenis Proyek       | Learning Management System (LMS)                             |
| Framework          | Svelte-kit                                                   |
| Database           | PostgreSQL                                                   |
| CLI                | sv                                                           |
| Status             | Proyek pribadi / in-progress                                 |
| Bahasa Pemrograman | JavaScript / TypeScript                                      |
| License            | MIT                                                          |

---

## Tujuan

Membuat sistem LMS modern untuk manajemen materi, pengguna, dan interaksi pembelajaran dalam lingkup pribadi atau kampus.
